# Cricket-Score-Tracker

## Description
This is a web-based cricket score tracking project.
This application allows users to record cricket match data and store match information in a database.
The goal of the project is to build a complete cricket scoring system capable of storing ball-by-ball match details.

## Features
- Store cricket match records
- Record innings data
- Track total runs scored by both teams
- Create a match history

## Current Stage
The project is currently in its initial stage.
At present, the application is capable of:
- Creating a new match
- Recording total runs for both innings
- Storing match data in the database

## Aim of the Project
The long-term goal of this project is to develop a complete cricket scoring system capable of storing:

- Ball-by-ball data
- Player-by-player statistics
- Over-by-over match details
- Innings-by-innings scorecards

## Tech Stack
- Python
- FastAPI
- MySQL
- HTML,CSS
